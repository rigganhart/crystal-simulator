The Nitty Gritty

There are alot of assumptions made in the programming of this site about how COC works without seeing the actual source code of COC. please keep in mid these are all just assumptions based on game logic.

The idea behind all this is that there are 3 tiers of rewards when you open a PHC. for lack of better terms I will use basic, rare, and super rare. Ex: 2star = basic; 3star = rare; 4star = super rare. When you open a crystal, you have a chance at getting one of the rewards. This logic exists in pretty much all free to play games.

Some basic code for the above mentioned logic:

if(randomNumber >= 0.66) { crystal = "2star"} else if(randomNumber < 0.66 && randomNumber>= 0.33 ) {crystal = "3star"} else {crystal = "4star"}

The above code is for an equal (33%) chance to get a 2, 3, or 4 star

Explanation

First you need a random number, in JS there is Math.random() which gives a random number between 0 and 1. This gives a nice random percentage ex: 0.25678 it goes to 16 decimal points so it could easily get to a super super rare percentage roll. ex: 0.0005

Try It Yourself

Right Click and select inspect. In the window that opens up click on Console. In the white space copy and paste: Math.random();

This is the "roll" you get a different number every time. I set a variable so it can be used in various functions

next you use an if statement: If the roll is greater than or equal to 0.66 the crystal (another variable) is equal to 2star, If the roll is less than 0.66 and greater than or equal to 0.33 the crystal is equal to 3star, If the roll is anything else the crystal is equal to 4star. This sets the percentage ranges at 33%

Obviously the percentages in COC are not equal so I adjusted them to my educated guess assumptions. I used 85% for 2star, 13% for 3 star, and a measely 2% for 4 star.
